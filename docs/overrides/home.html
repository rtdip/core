Real Time Data Ingestion Platform

RTDIP (Real Time Data Ingestion Platform) is a scalable, industry-standard solution for the intake and processing of data from a vast array of sources. These include time series, alarms, video, photos, audio, and input from hardware including IoT Sensors, Robots, and Drones. Fully open-source, it’s capable of taking in data from over 3 million sensors across the globe.

What to expect

Pipelines

RTDIP has been built to simplify ingesting and querying time series data; it has batch ingestion pipelines to match the requirements of the data source and needs of the data consumer.

Queries

Execute both macro and micro level queries on any dataset, enabling as granular and holistic view as required.

Deployment

A full SDK, written in Python, is provided, enabling ease of build and deployment of your implementation.

MultiCloud

Storing your data in multiple clouds? RTDIP works seamlessly across these resources.

Open Source

RTDIP is compliant with the Apache Open Source license, enabling maximum flexibility in adaptation and re-use.

Artificial Intelligence

Tested with ChatGPT, RTDIP can provided datasets for intelligent curation and query by AI agents.

Powerful ingestion pipelines

Query builder

Use RTDIP’s Query Builder feature to interrogate data in any delta table.

RAW +

A function which returns raw data by querying Databricks SQL Warehouse using a user-specified connection.

Interpolate +

This function forward fills, backward fills or linearly interpolates the resampled data depending upon the user’s specified method.

Resample +

A function in Spark which resamples data by querying Databricks SQL warehouses using a user-specified connection and authentication method.

Time Weighted Average +

This function receives a dataframe of raw tag data and creates a time weighted average.

Interpolation at Time +

A function which computes the linear interpolation at a specific time based on the points before and after.

Circular Average +

A function that receives a dataframe of raw tag data and computes the circular mean for samples in a range.

Circular Standard Deviation +

This function receives a dataframe of raw tag data and computes the circular standard deviation for samples assumed to be in the range.
